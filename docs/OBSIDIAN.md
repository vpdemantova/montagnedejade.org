# _esbocos — Vault Obsidian de referência

Esta pasta contém as notas e esboços do Portal Solar escritas no Obsidian.

## Como usar

1. Copie o conteúdo do seu vault Obsidian para dentro desta pasta
2. Diga ao Claude: **"leia o vault e comece a implementar"**
3. O Claude irá:
   - Ler todos os `.md` recursivamente
   - Cruzar com o código existente
   - Implementar o que está planejado mas ainda não existe
   - Ajustar o que está diferente do planejado

## Notas

- Esta pasta está no `.gitignore` — não será commitada
- Quando tudo estiver na plataforma, esta pasta pode ser deletada
- Use tags `#prioridade-1`, `#prioridade-2` nos seus arquivos para indicar ordem de implementação
